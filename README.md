# Apollo Graphql Server + Typescript

This apollo-graphql-server project comes with typescript already setup and ready!

A simple file structure is also present:
- `index.ts` -> the entry point for the server
- `server.ts` -> server initialization, exports a function that creates the server instance, allowing testing!
- `schema.ts` -> graphql schema
- `resolvers.ts` -> graphql resolvers
- `types.d.ts` -> typescript types
